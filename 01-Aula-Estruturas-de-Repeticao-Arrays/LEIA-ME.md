# Aula 01 - Estruturas de Repetição e Arrays

## 📚 O que você vai aprender nesta aula

Agora que você domina conceitos básicos, está na hora de trabalhar com **múltiplos dados** e **repetições**. Nesta aula você aprenderá a criar loops e manipular arrays.

### 🎯 Tópicos Principais

1. **Loops - Estruturas de Repetição**
   - `for` - Loop com controle de iterações
   - `while` - Loop condicional
   - `do-while` - Loop que executa pelo menos uma vez
   - `for-each` - Iteração sobre arrays e coleções

2. **Arrays**
   - Declaração e inicialização
   - Acessar elementos
   - Comprimento (length)
   - Iteração sobre arrays

3. **Estruturas de Repetição Aninhadas**
   - Múltiplos loops dentro de loops
   - Padrões e estruturas visuais

### 🔍 Conceitos Importantes

**For vs While:**
- Use `for` quando sabe quantas iterações fará
- Use `while` quando a condição é desconhecida

**Arrays:**
- Todos elementos devem ser do mesmo tipo
- Índices começam em 0
- `array.length` retorna o número de elementos
