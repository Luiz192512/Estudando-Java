# Aula 00 - Conceitos Básicos, Operadores e Condicionais

## 📚 O que você vai aprender nesta aula

Esta é a aula **introdutória** para java! Aqui você aprenderá os conceitos fundamentais que servem de base para todo o resto da linguagem.

### 🎯 Tópicos Principais

1. **Variáveis e Tipos de Dados**
   - Declaração de variáveis
   - Tipos primitivos (int, double, boolean, String, etc.)
   - Conversão de tipos

2. **Operadores**
   - Operadores aritmético (+, -, *, /, %)
   - Operadores de comparação (==, !=, <, >, <=, >=)
   - Operadores lógicos (&&, ||, !)
   - Operador ternário (?:)

3. **Estruturas Condicionais**
   - Estrutura if/else/else if
   - Estrutura switch/case
   - Operador ternário para condições simples

### 📝 Arquivos da Aula

| Arquivo | Descrição |
|---------|-----------|
| `Exemplo.java` | Primeiros exemplos com sintaxe básica |
| `Operadores.java` | Demonstração de operadores matemáticos |
| `Condicional.java` | Exemplos com estruturas if/else |
| `Switch.java` | Uso da estrutura switch/case |
| `OperadorTernario.java` | Operador ternário em ação |

### 🧪 Como executar os exemplos

1. Compile cada arquivo com:
   ```bash
   javac Exemplo.java
   ```

2. Execute com:
   ```bash
   java Exemplo
   ```

### 💪 Desafios Propostos

- [ ] Criar um programa que lê 2 números e exibe qual é o maior
- [ ] Validar se um número é positivo, negativo ou zero
- [ ] Implementar um menu usando switch/case
- [ ] Calcular o IMC usando operadores

### 🔗 O que vem depois

Depois de dominar estes conceitos, você estará pronto para:
- **Aula 01**: Estruturas de Repetição com Loops e Arrays
- Trabalhar com múltiplos dados de forma eficiente
